    运单管理专家

功能：
识别快递运单中的条形码，支持批量识别并导入文件名为解码值。
对快递运单进行批量管理和储存。

使用方法：
1、点击“单文件识别”按钮，选择一张带有条形码的运单相片，计算机将进入识别状态，识别结果显示在右侧文本框中。
2、点击“文件夹识别”（发件或签单）按钮，选择一个文件夹，计算机将识别该文件夹下所有相片，识别结果显示在右侧文本框中，并自动压缩和命名相片，导出到output文件夹中。
3、在“参数设置”窗口中可设置扫描精度系数和识别明暗系数，可对部分识别率较低的相片识别前适当修改参数。
4、在“参数设置”窗口中可设置待识别的条形码的类型，选择对应的快递公司名称即可。
5、通过查询菜单，可查询运单图片及指定日期的运单数。
6、在用户菜单可进行用户的设置和登陆，登陆后操作生成的图片文件将存入自己的隐私文件夹中，并对图片进行加密，只有本人可以浏览查看。
7、使用用户菜单下的“批量图片解码”功能可以将本人的加密图片批量转换为正常图片，方便在其他场合应用。
8、连接系统配套的数码扫描设备，点击“单次扫描”按钮，能直接完成通过扫描设备获取图片，随之进行识别，并自动储存的一体化功能。点击“连续扫描”按钮，能按照指定的拍摄间隔连续扫描，大大提高工作效率。
9、批量识别或扫描仪识别后，选择操作菜单中的“导出运单录入信息”功能可将之前所识别的信息导出到excel表格（必须安装有MSOffice软件），在选项菜单中的“运单录入信息设置”功能可以修改导出数据的参数。
10、批量识别或扫描仪识别后，选择操作菜单中的“上传已处理图片”功能可将之前识别处理后生成的图片上传至互联网，上传时耐心等待，支持续传。


注：
1、应尽量保持待识别条形码的平整
2、条码的像素尽量保持在400*200以上易于识别
3、尽量避免拍摄相片的光线过暗
4、被撕裂的条码识别率较低
